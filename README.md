# Hand Gesture Recognition for Touchless Control

A real-time gesture recognition system using computer vision.  
Controls system actions (volume, slides, media) via hand gestures — no touch required.

## Stack
- **Python + MediaPipe + OpenCV** — gesture detection pipeline
- **FastAPI** — backend REST + WebSocket server
- **React** — frontend UI with live webcam feed
